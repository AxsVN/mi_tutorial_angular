Ejemplo de cómo crear un contador con botones para incrementar y decrementar el valor.


1. Componente TypeScript (app.component.ts):


import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  contador: number = 0;

  incrementar() {
    this.contador++;
  }

  decrementar() {
    this.contador--;
  }
}

2. HTML del Componente (app.component.html):


Contador: {{ valor_contador }}

Explicación:

  • TypeScript: Define un componente con una variable contador inicializada a 0, y dos métodos incrementar() y decrementar() para modificar su valor.

  • HTML: Muestra el valor del contador y tiene dos botones. Al hacer clic en los botones, se llaman a los métodos incrementar() y decrementar() para aumentar o disminuir el valor del contador.


    Este ejemplo es muy sencillo y sirve como una introducción rápida a Angular para crear una funcionalidad básica. No visualizamos el valor de CONTADOR, ya que esto no deja de ser un simple tutorial.
