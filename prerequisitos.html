Herramientas y Requisitos

Para comenzar a trabajar con Angular, asegúrate de tener los siguientes prerrequisitos:

  • Node.js (versión 12.x o superior)
  • npm (Node Package Manager)
  • Editor de código, como Visual Studio Code
  • Conocimiento básico de JavaScript y HTML
