Utilización de Angular: Crear una Aplicación Base

En esta sección, vamos a explicar cómo crear una aplicación base en Angular y cómo visualizar los resultados de su ejecución.


1. Crear una Aplicación Base en Angular

Para crear una aplicación base en Angular, sigue estos pasos:


Paso 1: Crear el Proyecto

Abre la terminal o línea de comandos y usa Angular CLI para crear un nuevo proyecto. En este caso, llamaremos al proyecto mi-aplicacion:

ng new mi-aplicacion

Angular CLI te pedirá algunas opciones, como si deseas incluir el enrutamiento y qué tipo de hojas de estilo utilizar (CSS, SCSS, etc.).


Paso 2: Navegar al Proyecto

Una vez creado el proyecto, navega al directorio del proyecto:

cd mi-aplicacion

Paso 3: Crear un Componente (Opcional)

Angular crea un componente principal llamado AppComponent de forma predeterminada, que es el que se utiliza como punto de partida para la aplicación. Si deseas crear un nuevo componente, puedes usar el siguiente comando:

ng generate component mi-componente

Este comando crea un nuevo componente llamado mi-componente, y se agregará automáticamente a tu proyecto.


2. Visualizar los Resultados de la Ejecución

Para visualizar la aplicación en tu navegador, sigue estos pasos:


Paso 1: Iniciar el Servidor de Desarrollo

Una vez que hayas creado el proyecto y configurado los componentes necesarios, usa el siguiente comando para iniciar el servidor de desarrollo:

ng serve

Este comando compila el proyecto y arranca un servidor local de desarrollo.


Paso 2: Abrir el Navegador

El servidor de desarrollo estará ejecutándose en el puerto 4200 por defecto. Abre tu navegador y accede a:

http://localhost:4200

Al hacerlo, verás la aplicación en ejecución. Si todo está bien configurado, deberías ver la página inicial con el contenido predeterminado del componente principal AppComponent.


Resumiendo lo comentado...

  • Crear una aplicación base: Usas ng new para generar un nuevo proyecto Angular.
  • Crear componentes (opcional): Angular CLI te permite crear fácilmente componentes con el comando ng generate component.
  • Visualizar la ejecución: Usa ng serve para iniciar el servidor local y abre http://localhost:4200 en el navegador para ver los resultados.
