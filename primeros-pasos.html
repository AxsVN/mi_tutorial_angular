Primera Aproximación en Angular: "Hola Mundo"

Para realizar una primera aproximación en Angular con el clásico ejemplo de "Hola Mundo", sigue estos pasos:


1. Crear un Proyecto Angular

Primero, asegúrate de tener Angular CLI instalado y crea un nuevo proyecto con el siguiente comando:

ng new hola-mundo

Este comando creará un nuevo proyecto llamado hola-mundo y te pedirá que elijas algunas configuraciones, como si deseas incluir Angular Routing y qué estilo deseas usar (CSS, SCSS, etc.).



2. Navegar al Directorio del Proyecto

Una vez creado el proyecto, navega al directorio del proyecto recién creado:

cd hola-mundo


3. Crear un Componente "Hola Mundo"

Angular crea un componente principal de manera predeterminada llamado AppComponent. Para personalizarlo y mostrar "Hola Mundo", sigue estos pasos:

  • Abre el archivo src/app/app.component.ts.

  • Dentro de este archivo, modifica la propiedad title de la siguiente forma:

    • 
	import { Component } from '@angular/core';

	@Component({
	  selector: 'app-root',
	  templateUrl: './app.component.html',
	  styleUrls: ['./app.component.css']
	})
	export class AppComponent {
	  title = 'Hola Mundo';
	}


    4. Modificar la Vista del Componente

    Abre el archivo src/app/app.component.html y reemplaza su contenido con lo siguiente:

    <h1>{{ title }}</h1>

    La doble llave {{ title }} es la vinculación de datos en Angular, lo que significa que el valor de la propiedad title se mostrará dentro del elemento <h1>.



    5. Ejecutar la Aplicación

    Para ver el resultado, regresa a la terminal y ejecuta el siguiente comando:

    ng serve

    Este comando iniciará el servidor de desarrollo y podrás ver la aplicación en tu navegador accediendo a: http://localhost:4200


    Verás la frase "Hola Mundo" en pantalla.
