Pasos para la instalación

Para instalar Angular en tu entorno local, sigue estos pasos:

  • 1. Asegúrate de tener Node.js y npm instalados.
  • 2. Ejecuta el siguiente comando en tu terminal: npm install -g @angular/cli
  • 3. Crea un nuevo proyecto con ng new my-app
